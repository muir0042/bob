What is PunyCode?

Punycode is a way to decode Unicode characters (UTF-8 - other special character like é) into regular ASCII characters so they can be read by computers in instances where Unicode special characters are not allowed ex domain names.

Not to be confused with code puns

Why use PunyCode?

Let’s say we have an RPG game where every character you create gets their own website. If I create a character named ‘Conan’ my website will be ‘bestgame-conan.com’ (as an example). The problem is that there are always people in RPG games who want games like ‘Conan’, ‘Legolas’, ‘Thorin’, ‘Shadow’ or any number of cliched names, and they will do anything they can to get that name. You end up with names like ‘xLégôlåŠx’ ‘ŽeÜs’ and other names that are impossible to type. These names also could not be used in domain names. This is when we would use our Punycode.

How to Install

First we need to go into node and install the Punycode module. Punycode now comes core with node.js.
Type:
npm install punycode
This will install punycode into node.
Now we can go into node and start taking advantage of some new functions punycode gives us.

How to Use

Let’s use punycode to encode our ‘xLégôlåŠx’ name:
punycode.encode(‘xLégôlåŠx’);
Should produce:
‘xLglx-praz3g81c’
Now if we wanted to use our punycode function to create that domain for this person’s profile we would type in this
punycode.toASCII(‘bestgame-xLégôlåŠx.com’);
will produce:
‘xn--bestgame-xLglx-xib5d2kw8e.com’
Not the prettiest domain but now it no longers contains the special characters. That’s what they get for having stupid characters in their name.
